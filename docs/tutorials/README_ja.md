## 利用方法
1. 第1章で設計思想（ペイロード優先、フェイルファスト、宣言的配線）を理解します。
2. 第2章の最小フローをローカル環境で実装します。
3. 第3章と第4章で分岐・ループ・ジョインを取り入れて段階的に拡張します。
4. 第5章の内容を参考に、設定管理・テスト・運用の体制を整えます。
